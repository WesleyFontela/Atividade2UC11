# Sistema de Leilões

Este projeto é um sistema informatizado para gerenciamento de uma casa de leilões.  
Ele permite cadastrar produtos, consultar a lista de itens disponíveis e realizar vendas.

## Funcionalidades principais
- **Cadastro de produtos**: inserir nome e valor de novos itens.
- **Consulta de produtos**: visualizar lista com ID, nome, valor e status.
- **Venda de produtos**: realizar a venda informando o ID do item.
- **Consulta de vendas**: acompanhar os produtos já vendidos.

## Tecnologias utilizadas
- Java (Swing para interface gráfica)
- MySQL (banco de dados relacional)
